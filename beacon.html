Sociocast Core Beacon API

Purpose

The Sociocast Beacon API is primarily used for entity observations via ad impressions and/or email campaigns. It is useful when you do not have access to JavaScript or other any other server-side languages. The Sociocast Beacon API is an invisible 1Ã—1 pixel image that clients can easily embed in emails or display advertising campaigns. The HTTP Request for the image registers an /entity/observation call in Sociocast's platform.
